cloudwatch-logs-cli
===========

_*CLI for AWS CloudWatch logs*_

About
--------

CloudWatch's Web UI is unusable.  Instead use this and not go crazy.


Installation
--------

    python3 ./setup.py build
    python3 ./setup.py install


Compatibility
--------

* Python 3.10+


Example Usage
--------

```shell
$ cloudwatch-logs /aws/lambda/my-log-group
```
